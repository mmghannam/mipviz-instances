# MIP/LP Instance Collection

This repository hosts a clean, versioned collection of Mixed-Integer and Linear Programming instances for use with [mipviz](https://github.com/mmghannam/mipviz) and other optimization tools.

The instances are stored in their original format (e.g., `.mps.gz`).

## Collections

- **MIPLIB 2017**: The benchmark instance set from [MIPLIB 2017](https://miplib.zib.de/).

## Usage

You can access the raw files directly via `raw.githubusercontent.com`. For example:

`https://raw.githubusercontent.com/mmghannam/mipviz-instances/main/miplib2017/10teams.mps.gz`
