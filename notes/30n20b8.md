# About this instance

`30n20b8` is a multi-mode resource-leveling problem with precedence
constraints, introduced by Coughlan, Lübbecke, and Schulz (2010). The
objective is to smooth resource usage over a scheduling horizon given
task precedences and alternative execution modes.

## Structure

The instance has a **block-angular structure** once decomposed — the
precedence and set-partitioning constraints form independent blocks
linked by a small number of general-linear coupling rows. This makes it
a natural fit for Dantzig–Wolfe reformulation.

## Techniques that help

- **Column generation / branch-and-price.** The authors' original
  approach — each pricing subproblem corresponds to a single task's
  mode assignment.
- **Aggressive presolve.** SCIP's `dualinfer` and clique-based fixings
  remove roughly half of the binaries before the root LP.
- **Symmetry handling.** There's meaningful permutation symmetry among
  identical modes; `symmetry/addconsstiming` in SCIP helps.

## References

- Coughlan, E., Lübbecke, M., & Schulz, J. (2010).
  *A Branch-and-Price Algorithm for Multi-mode Resource Leveling.*
  In: Experimental Algorithms, LNCS 6049, pp. 226–238. Springer.

## Trivia

Even though the instance is classified **easy**, it took several
years and a custom branch-and-price implementation before it was
first solved to proven optimality.
